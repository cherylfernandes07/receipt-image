How to use it

  1. Scan the QR code with your phone camera.
  2. Open the link and launch rCapture in your browser.
  3. Use Camera mode on mobile for live capture with mic permission.
  4. Use Upload mode on desktop to pick an image file and preview it.
  5. Switch modes anytime with Auto Detect / Use Camera / Use Upload.
